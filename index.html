136 Days of Functions

What is 136 Days of Functions?

136 Days of Functions was created to be a personal reference and blog, as I go through and document a use case for the (at present) 136 built in prototypal functions in Javascript. I thought, what better way to deepen my knowledge of Javascript, than to write a little post on each of the built in functions within Javascript and explain a use case or two for each.

While this is primarily a personal reference, I have typed it up as though there is an audience also using it as a reference. At the moment, it's only the first week, so I'm sure things will change greatly as this project evolves, and hopefully other people will contribute.
