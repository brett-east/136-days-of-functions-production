toExponential()

Overview

This method is part of the Number Object prototype and can be called on any object that is a Number, it converts a number to be displayed as an exponent. An exponent is displayed with e+# with # representing the number of places to move a decimal point. Read more about exponential numbers here: https://en.wikipedia.org/wiki/Scientific_notation.

Syntax

number.toExponential( [fractionDigits] );

Parameters

The toExponential() method takes one optional parameter called 'fractionDigits'. 'fractionDigits' is a whole number, specifying the number of digits after the decimal point to display before displaying the exponential.

If 'fractionDigits' is not specified, then the number of digits will be the number need to represent the value uniquely.

Examples

These examples are taken from the Mozilla Developer Network.

var numObj = 77.1234;

console.log(numObj.toExponential());  // logs 7.71234e+1
console.log(numObj.toExponential(4)); // logs 7.7123e+1
console.log(numObj.toExponential(2)); // logs 7.71e+1
console.log(77.1234.toExponential()); // logs 7.71234e+1
console.log(77 .toExponential());     // logs 7.7e+1
